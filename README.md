# nexmo-ng-sms
1. Download [ngrok](https://ngrok.com/download)
2. `$ ./ngrok http 3000`
3. `$ npm install express body-parser pusher`
4. `$ node server.js`
5. `$ cd nexmo-sms`
6. `$ npm install -g @angular/cli`
7. `$ npm install`
8. `$ ng serve`
9. Send an SMS to +447520632539 and see the magic happen
